# 通知样式自定义说明

## 已实现的功能

### 1. 主题一致性
- ✅ 深色背景渐变（#1f2937 到 #111827）
- ✅ 与界面风格统一的配色方案
- ✅ 半透明毛玻璃效果（backdrop-filter）
- ✅ 精致的边框和阴影

### 2. 四种通知类型样式

#### 成功通知（Success）
- 左侧绿色边框（#10b981）
- 绿色渐变背景
- 绿色图标

#### 错误通知（Error）
- 左侧红色边框（#ef4444）
- 红色渐变背景
- 红色图标

#### 警告通知（Warning）
- 左侧橙色边框（#f59e0b）
- 橙色渐变背景
- 橙色图标

#### 信息通知（Info）
- 左侧蓝色边框（#3b82f6）
- 蓝色渐变背景
- 蓝色图标

### 3. 交互效果
- ✅ 平滑的滑入动画
- ✅ 悬停时的阴影增强和轻微上浮
- ✅ 关闭按钮的悬停效果
- ✅ 最多同时显示3个通知

### 4. 全局配置
```typescript
const [api, contextHolder] = notification.useNotification({
  placement: 'topRight',  // 右上角显示
  top: 24,                // 距离顶部24px
  duration: 3,            // 默认显示3秒
  maxCount: 3,            // 最多同时3个通知
});
```

## 使用示例

### 基础用法
```typescript
// 成功通知
api.success({
  message: '操作成功',
  description: '您的操作已成功完成',
  duration: 2,
});

// 错误通知
api.error({
  message: '操作失败',
  description: '发生了一些错误，请重试',
  duration: 2,
});

// 警告通知
api.warning({
  message: '警告',
  description: '请注意某些限制条件',
  duration: 3,
});

// 信息通知
api.info({
  message: '提示',
  description: '这是一条提示信息',
  duration: 2,
});
```

### 高级用法（JSX 内容）
```typescript
api.success({
  message: '聊天室创建成功',
  description: (
    <div>
      <p>聊天室ID: <strong>123456789</strong></p>
      <p className="text-xs text-gray-400 mt-1">✓ ID已复制到剪贴板</p>
    </div>
  ),
  duration: 4,
});
```

## 样式特点

### 颜色方案
- 主背景：深灰色渐变
- 文字颜色：浅灰白色（#f3f4f6, #d1d5db）
- 强调文本：蓝色（#60a5fa）
- 边框：半透明灰色（#374151）

### 视觉效果
- 圆角：12px
- 阴影：多层阴影营造深度感
- 动画：从右侧滑入（0.3s）
- 悬停：向上平移2px + 阴影增强

### 响应式
- 最大宽度：400px
- 自适应内容高度
- 移动端友好

## 自定义选项

如需进一步自定义，可在 `src/index.css` 中修改：

1. **颜色调整**：修改 `.ant-notification-notice-success` 等类的颜色值
2. **尺寸调整**：修改 `max-width`、`padding` 等属性
3. **动画效果**：调整 `@keyframes notificationSlideIn` 动画
4. **阴影效果**：修改 `box-shadow` 属性

## 技术实现

### CSS 自定义类
所有样式都在 `src/index.css` 中定义，使用 `!important` 覆盖 Ant Design 默认样式。

### 主要 CSS 类
- `.ant-notification-notice` - 通知容器
- `.ant-notification-notice-success` - 成功类型
- `.ant-notification-notice-error` - 错误类型
- `.ant-notification-notice-warning` - 警告类型
- `.ant-notification-notice-info` - 信息类型
- `.ant-notification-notice-message` - 标题
- `.ant-notification-notice-description` - 描述内容

### 兼容性
- ✅ Chrome/Edge（完整支持）
- ✅ Firefox（完整支持）
- ✅ Safari（部分 backdrop-filter 可能需要前缀）
- ✅ 移动端浏览器
